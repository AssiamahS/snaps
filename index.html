data

Doctors, dentists, pharmacists. Refreshed monthly against the official CMS NPPES dump.

speed

~5ms indexed lookups

State + city + last name + taxonomy. Optimized for high-frequency credentialing checks.

json that fits

Flat rows, ISO dates, one column per field. No stringified CSV, no legacy encodings.

Query Parameters

state ISO 3166-2 (e.g. CA)
city Canonical city name
taxonomy 10-char specialty code
limit 1-100 (Default 20)
