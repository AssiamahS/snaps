Standard HTTP status codes. Every list endpoint supports cursor-based pagination.

Error Schema

{
  "error": {
    "code": "invalid_taxonomy",
    "message": "The taxonomy code provided is not a valid CMS provider taxonomy code."
  }
}

Query Parameters

state 2-letter U.S. state code, e.g. NJ, CA, TX
city City name, case-insensitive
taxonomy 10-character CMS provider taxonomy code
limit 1–100, default 20
cursor Pagination cursor returned from the previous response
